Welcome to SPARTACUS10's documentation!
=====================================
The SPARTACUS10 package includes functions to perform spatial hierarchical 
agglomerative clustering (SHAC), including SPARTACUS (SPAtial hieRarchical 
agglomeraTive vAriable ClUStering), as well as spatially constrained ensemble 
clustering (SEC). These functions are especially designed to cluster neuroimaging 
data. Moreover, implementations of the silhouette coefficient (SC) for large data, 
the simplified silhouette coefficient (SSC) and spatial adaptations thereof are 
included as well.
